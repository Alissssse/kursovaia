Примечания

count() используется для получения точного количества объектов в базе данных. Вместо загрузки всех объектов и подсчета их в Python, Django выполняет оптимизированный SQL-запрос SELECT COUNT(*).

exists() проверяет наличие хотя бы одного объекта в базе данных. Это быстрее, чем count() > 0, так как выполняет запрос SELECT 1 ... LIMIT 1 вместо подсчета всех записей.
